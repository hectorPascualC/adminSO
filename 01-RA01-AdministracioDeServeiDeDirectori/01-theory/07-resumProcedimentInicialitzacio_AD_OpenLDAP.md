# **RESUM GENERAL del Procediment d'Inicialitzacio d'AD i OpenLDAP**

* AD DS o OpenLDAP = servei de directori
* Instal·lació AD → rol AD DS → promoció
* Crear domini → configuració DNS → OU → usuaris → grups → equips
* Confianzas → connectar dominis i boscos
* LDAP → slapd → configuració base → phpLDAPadmin
* Finalitat: centralitzar autenticació i permisos
